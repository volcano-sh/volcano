# Volcano Documentation

Welcome to the Volcano documentation!

Volcano is a batch system built on Kubernetes. It provides a suite of mechanisms that are commonly required by many classes of batch and elastic workload including: machine learning/deep learning, bioinformatics/genomics and other "big data" applications.

## What is Volcano?

Volcano is a Kubernetes native batch scheduling framework. It provides:

- **Gang Scheduling**: Co-scheduling of multiple pods
- **Job Management**: CRD-based job abstraction
- **Queue Management**: Resource sharing and fairness
- **Plugin Framework**: Extensible scheduling plugins
